# Radio

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | ---- | --- |
| className | 设置类名 | `string` | `-` |
| name | name | `string` | - |
| value | radio标示 | `string` | - |
| checked | 当前是否选中 | `boolean` | false |
| disabled | 是否禁用 | `boolean` | false |
| color | 颜色 | `string` | - |
