# Tag

## API

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | ---- | --- |
| className | 设置类名 | `boolean` | `false` |
| name | name | `string` | - |
| size | 大小 | `small` | - |
| active | 激活状态 | `boolean` | - |
| circle | 是否圆角 | `boolean` | - |
| disabled | 禁用 | `boolean` | - |
| customStyle | 自定义样式 | `any` | - |
| onClick | onClick | `(）=> void` | `-` |
